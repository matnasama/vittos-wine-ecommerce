# Vittos Wine - Ecommerce App

This is a fullstack ecommerce project for a wine shop, built with:

- **Frontend:** React + Vite + Material UI
- **Backend:** Node.js + Express + MySQL + JWT
- **Auth:** Secure login/register, token-based auth
